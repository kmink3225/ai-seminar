# qa — 질의응답

## 목적

사용자 질문에 **내부 문서 근거** 를 기반으로 답변한다. 파일은 변경하지 않는다.

## 전제

`guides/search.md` 를 먼저 로드·실행한 후 이 가이드를 따른다.

## Step

1. **검색** — `search.md` Step 을 실행해 근거를 모은다.
2. **근거 충분성 판정**
   - 충분 (L1 또는 L2 에서 직접 근거 확보) → Step 3 진행
   - 부족 → "내부 문서에 근거 부족. 외부 지식 기반 일반 답변이 필요하면 알려 달라." 로 보고하고 종료
3. **통합 답변 작성**
   - 구조: 결론 → 근거 → 한계
   - 결론: 2~4 문장
   - 근거: 불릿, 각 항목 끝에 `(path:line)` 인용
   - 한계: 내부 문서로 확인되지 않는 부분·반대 견해 명시
4. **추가 읽을거리** — 관련 내부 문서 2~3 건을 링크로 제시한다.

## 답변 템플릿

```
## 결론
…

## 근거
- … (references/md/xxx.md:42)
- … (topics/rag/reranking.md:15)

## 한계
- …

## 추가 읽을거리
- [제목](경로)
```

## 금지

- 외부 지식으로 보강한 부분을 근거 인용처럼 위장
- 질문 재진술로 분량 늘리기
