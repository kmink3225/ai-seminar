# Topics

주제별 **상시 참고 문서** 폴더다. 세미나 날짜와 무관하게 계속 참조되는 **개념 본문**(정의, 원리, 예제, 비교)을 보관한다.

## 역할

- 발표를 위해 작성한 개념 설명 md 파일은 여기에 둔다.
- 세미나 Q&A·논의 내용도 해당 주제 문서에 포함한다.
- 즉 topics/ 가 개념 본문과 세미나 기록을 모두 담당하는 유일한 지식 폴더다.

## 하위 폴더

| 폴더 | 범위 |
|------|------|
| `engineering/` | 프롬프트 → 컨텍스트 → 하네스 엔지니어링 층위 전체 |
| `rag/` | 검색 증강 생성 — 임베딩, 벡터 스토어, 청킹, 리랭킹 |
| `agents/` | 에이전트 구현 — tool use, planning, multi-agent, 프레임워크 |
| `models/` | 모델 비교, 파인튜닝, 컨텍스트 관리, 비용/성능 분석 |
| `tools/` | 개발 도구, SDK, 플랫폼 — Claude Code, Cursor, MCP, API 활용 |

## 파일 규칙

- 파일명: `kebab-case.md`
- 상단에 YAML 프론트매터 권장 (`title`, `author`, `date`, `tags`)
- 하위 폴더에 맞는 주제에 배치한다. 여러 주제에 걸치면 주된 주제 폴더에 넣고 다른 폴더에서 링크한다.
